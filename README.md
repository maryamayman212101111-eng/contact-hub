
Contact Hub
fully functional contact management application built with React.js. This app allows users to store, retrieve, and manage their contact details efficiently.
 • Features
- Add new contacts with name, phone, and email.
- Edit existing contact details.
- Delete contacts with confirmation.
- Form validation to ensure data accuracy.
- Responsive design for mobile and desktop.
- Local storage support to save
   • 🛠 Technologies Used

- **java script**
- **bootstrap CSS** (Styling)
- **Git & GitHub** (Version Control)
